ナビゲーションバーは.navbarクラスで生成される ナビゲーション横配置から縦配置に変更する際の 画面サイズは.navbar-expand-xl|lg|md|smで設定する .navbar-expandがなければ最初から縦配置になる。
ナビゲーションにリンクを追加したい場合はulタグにクラスnavbar-navを適用し、 その子にnav-itemクラスを追加したliタグ、さらにその子のaタグ(またはspanタグ)にnav-linkクラスと、 href属性(リンク先を追加すること)を追加すればよい。 
          <nav class="navbar navbar-expand-sm bg-light">
          <ul class="navbar-nav">
          <li class="nav-item">
          <a class="nav-link" href="#">google</a>
          </li>
          <li class="nav-item">
          <a class="nav-link" href="#">yahoo</a>
          </li>
          <li class="nav-item">
          <a class="nav-link" href="#">youtube</a>
          </li>
          <li class="nav-item">
          <a class="nav-link" href="#">twitter</a>
          </li>
            </ul>
            </nav>

navタグのクラスにbg-colorを追加すればバーの背景色を変更できる。
色は次の表の通り。bg-darkおよびbg-lightは文字色が重ならないようにする。
class color
.bg-primary Primary
.bg-success Success
.bg-info Info
.bg-warning Warning
.bg-danger Danger
.bg-secondary Secondary
.bg-dark Dark ※navbar-lightと併用
.bg-light Light ※navbar-darkと併用

ハンバーガーアイコンの生成にはbuttonタグを使用する。
ボタンタグにclass="navbar-toggler",data-toggle="collaspe" と、ハンバーガーアイコンにする対象のid属性をdata-target="#thetarget"で指定する。
アイコンのデザインはspanタグのクラスで指定する。
対象のナビゲーション(ul)はdivタグで囲み、divタグにはclass="cpllaspe navbar-collapseとid=thetargetを追加する。 
          <!--ボタンの生成-->
          <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#targettest">
          <span class="navbar-toggler-icon"></span>
          </button>

          <!--ナビゲーションバー側の指定-->      
          <div class="collapse navbar-collapse" id="targettest">
          <ul class="navbar-nav">
            ︙
          </div>
